배열

- 자바스크립트에서는 변수 선언 시 별도의 자료형 지정이 없었기 때문에 마찬가지로 배열 공간에도 어떤 자료형의 값들이던 간에 다 차곡차곡 담아낼 수 있음!! (자바의 ArrayList 와 유사)



* 배열의 선언

- 자바스크립트에서는 배열 선언 시 크기를 지정한 채로 선언하거나, 크기를 지정하지 않아도 선언이 가능하다.
- 즉, 배열 크기의 제약이 없다. (자바의 ArrayList 와 유사)



* Array 내장 객체의 메소드

1) indexOf(찾고자하는요소) : 배열에서 해당 요소가 위치해있는 인덱스를 반환



2) concat(배열명, 배열명, ...) : 여러개의 배열을 결합하고자 할 때 사용



3) reverse() : 배열에 담긴 값들을 역순으로 바꿔주는 메소드



4) sort() : 배열 안에 담긴 값들을 오름차순으로 정렬시켜주는 메소드



5_1) push(추가할요소) : 배열의 맨 뒤에 요소를 추가하고 배열의 크기를 반환
5_2) pop() : 배열의 맨 뒤 요소를 제거하고 그 제거된 요소를 반환



6_1) unshift(추가할요소) : 배열의 맨 앞에 (0번 인덱스) 요소 추가 후 배열의 크기를 반환
6_2) shift() : 배열의 맨 앞 (0번 인덱스) 요소를 제거 후 제거된 요소를 반환



7_1) slice(시작, 끝) : 배열 안의 요소들을 단지 추출해주는 메소드
7_2) splice(시작, 제거수, 추가값) : 배열의 요소를 추출해서 제거 및 추가, 추가값은 생략 가능



8) join(구분자) / toString() : 배열에 담긴 값들을 하나의 "문자열" 로 합쳐서 반환해주는 메소드
