함수 (function)

* 선언적 함수

        [ 표현법 ]

        function 함수명(매개변수, 매개변수, ... => 생략 가능) {

            해당 함수 호출 시 실행할 구문들;

            return 결과값;
        }

        - 선언적 함수 : 이름이 있는 함수로써, 이름이 있기 때문에 개발자인 내가 직접 호출이 가능한 함수
        - 매개변수는 생략 가능
        - 반환할 결과값이 없는 경우 return 구문 생략 가능
          (return 구문이 실행되는 시점에서 반환형이 정해지므로 함수 머리부에 반환형을 기술하지 않음)


* 익명 함수

        [ 표현법 ]

        변수 = function(매개변수, 매개변수, ... => 생략 가능) {

            소스코드;

            return 결과값;
        };

        - 익명 함수 : 특정 변수나 속성에 대입되는 값으로 취급하는 함수를 제시할 때 주로 사용됨
                     (주로 이벤트 핸들러 작성 시 사용)
        - 매개변수는 생략 가능
        - return 구문 또한 생략 가능
