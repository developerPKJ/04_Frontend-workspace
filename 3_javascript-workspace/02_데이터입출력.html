자바스크립트에서의 데이터 입출력

1. 데이터를 출력하는 기본 구문

1) alert("알림창에 출력할 문구")
2) console.log("개발자 도구 콘솔창에 출력할 문구")
3) document.write("웹페이지에 출력할 문구")
4) 선택한요소.innerHTML/innerText = "선택한 요소에 출력할 문구"



1) alert(알림창에 출력할 문구)



2) console.log(개발자 도구 콘솔창에 출력할 문구)



3) document.write(웹페이지에 출력할 문구)



4) 선택한요소.innerHTML/innerText = "선택한 요소에 출력할 문구"



2. 데이터를 입력받는 기본 구문(변수에 기록 가능)

1) 변수 = confirm("질문내용");
2) 변수 = prompt("질문내용");
3) 선택한요소.value;



1) confirm("질문내용");

- confirm 호출 시 "질문내용"과 "확인/취소" 버튼이 존재하는 알림창이 발생
- "확인" 버튼 클릭 시 true, "취소" 버튼 클릭 시 false 반환



2) prompt("질문내용");

- prompt 호출 시 "질문내용"과 "입력창", "확인/취소" 버튼이 존재하는 알림창이 발생
- 입력창에 입력한 값이 문자열 형태로 반환되고, "취소" 버튼 클릭 시 null 반환





    3) 선택한요소.value;
