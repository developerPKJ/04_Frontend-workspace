객체

* 객체 선언 및 호출

- 객체는 중괄호 {} 를 이용해서 생성하고, 중괄호 안에 이 객체에서 필요로 하는 속성들을 정의함
- 이 때, 속성명 : 속성값의 형태로 제시 (키 : 밸류)
- 속성값으로는 모든 자료형의 값들을 다 담을 수 있음 

        [ 표현법 ]

        var 변수명 = {
            속성명 : 속성값,
            속성명 : 속성값,
            ...
        };


* 객체 생성 특이케이스

객체 생성 시, 속성명을 제시할 때 반드시 문자열 형태 ("" 로 감싼 형태) 로 제시해야 하는 케이스가 종종 있음!!



* 객체에서의 반복문 활용

- 자바스크립트의 객체가 가지고 있는 모든 속성들에 대해서 순차적으로 접근하고자 한다면 반복문을 제시할 수 있다!!
- 단, 객체에는 인덱스 개념이 없기 때문에 일반 for 문 은 사용 불가하고, for in 문은 사용 가능함!!



* 객체의 속성값으로 함수를 제시하는 경우

- 객체의 속성 중 속성값이 함수 자료형인 경우 (속성값에 함수가 들어갈 수 있다 라는 뜻) 그 해당 속성을 "메소드 속성" 이라고 부른다.

- 객체명.메소드속성명(); 로 호출할 수 있다.
