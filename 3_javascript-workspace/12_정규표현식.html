정규표현식

- 정규표현식 (REGEX / REGEXP : Regular Expression) 이란, 특정 패턴을 가진 문자열을 찾거나 또는 찾아서 변경 (치환) 할 때 사용되는 형식 언어
- 정규표현식을 이용하면 문자열에 대해 특정 패턴 검사 또는 치환 시 복잡한 조건식을 제시할 필요가 없이 간단하게 처리 가능!!
- 자바스크립트 뿐만 아니라 자바나 오라클 등 다른 언어에서도 사용 가능 (단, 문법이 살짝 다를 수는 있음)



* 정규표현식 객체 생성 및 정규표현식 관련 메소드



* 메타문자

- 메타 문자를 이용해서 정규식으로 검사할 특정 패턴을 제시할 수 있다.
- 문자열이 해당 정규식으로 제시한 특정 패턴에 만족하는지 test() 로 검사하거나, replace() 를 이용해서 치환 또한 가능하다.



* 플래그문자

- /정규표현식/ 뒤에 작성하여 해당 정규식을 보조해주는 역할
- 여러개 제시 가능, 순서 상관 없음

i : 대소문자를 가리지 않고 비교를 수행하겠다. (ignore)
g : 문자열 내의 모든 패턴을 찾겠다. 즉, 전역으로 비교를 수행하겠다. (global)
m : 여러줄 비교를 수행하겠다. (multi)



* 정규표현식을 가지고 주민번호 확인
