정규표현식

- 정규표현식 (REGEX / REGEXP : Regular Expression) : 특정 패턴을 가진 문자열을 찾거나 대체하는 데 사용하는 문자열(형식 언어)
: 정규표현식을 이용하면 무자열에 대해 특정 패턴 검사 / 치환 시 복잡한 조건식을 생략가능
: 자바스크립트 뿐만 아니라 자바나 오라클 등 다른 언어에서도 존재(문법은 조금씩 다름)



정규표현식 객체 생성 및 관련 메소드



메타문자

- 메타 문자를 이용해서 정규식으로 검사할 특정 패턴을 제시할 수 있다
- 문자열이 해당 정규식으로 제시한 특징 패턴에 만족하는지 test()로 검사하거나, replace()로 치환 가능
