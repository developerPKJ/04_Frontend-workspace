window 용 객체

- window 객체는 자바스크립트의 최상위 내장 객체이며, 브라우저창 하나가 열릴 때 마다 내부적으로 하나씩 생성된다!! (브라우저 창 하나를 나타냄)
- window 객체의 하위 속성들 (자식들) 이 다양한데, 크게 BOM 계열과 DOM 계열로 나뉜다.

1. BOM (Browser Object Model) : 브라우저 창과 관련된 하위 속성객체들
location 객체, screen 객체, navigator 객체, history 객체 등
2. DOM (Document Object Model) : html 문서와 관련된 하위 속성객체
document 객체



* window 객체

- 자바스크립트의 가장 최상위 내장객체로써, 브라우저창 하나를 나타내는 객체임!!
- 브라우저창이 하나 열릴 때 마다 생성됨 

        브라우저창이 열릴 때 마다 내부적으로

        var window = {
            alert : function() {},
            confirm : function() {},
            prompt : function() {},
            console : {
                log : function() {},
                dir : function() {},
                ...
            },
            document : {
                write : function() {},
                getElementById : function() {},
                ...
            },
            history : {},
            location : {}, 
            ...
        };

        이런 식으로 객체가 하나씩 생성되는거임!!

HTML에 있는 각각의 태그들을 노드라고 부름
- 요소노드(element node) : 태그 그 자체를 의미
- 텍스트노드(text node) : 태그 사이에 기록되는 텍스트를 의미

HTML 요소들은 크게 두가지 분류로 나눌 수 있음 - 텍스트노드가 존재하는 요소 : 시작태그와 종료태그가 한쌍으로 이루어진 요소들 (div, p, span, button 등등) - 텍스트노드가 존재하지 않는 요소 : 오로지 시작태그로만 이루어진 요소들 (input, img, br 등등)
