변수와 자료형

변수

- 프로그래밍에서 메모리에 값을 담는 공간 (상자)
- 그 보관된 값은 매번 변할 수 있음!! (그래서 변수라고 부름)


* 변수 선언 시 var 뿐만 아니라 let, const 도 사용 가능

var, let, const 의 차이점

1) 변수 선언 방식



2) 스코프 (Scope : 이 변수를 사용 가능한 영역, "범위")



* 변수 선언 위치에 따른 전역변수 / 지역변수

        <script>
        
            var 변수명; // 전역변수 (global)

            변수명; // 전역변수 (global)

            function 함수명() {

                var 변수명; // 지역변수 (local)

                변수명; // 전역변수 (global)
            }

        </script>


* 자료형

- 자바스크립트에서는 변수 선언 시 자료형을 별도로 지정하지 않음
- 그렇다고 해서 자료형의 개념이 없는건 아님!!
- 변수에 대입되는 값 (리터럴) 에 따라서 알아서 자료형이 자동으로 결정됨
(대입 시점에서 자료형이 결정되는 것 : 타입추론)



자바스크립트의 자료형

  • 원시 타입 (primitive type)
    1. string (문자열)
    2. number (숫자)
    3. boolean (논리값)
    4. undefined (초기화가 안된 변수)
  • 참조 타입 (reference type)
    1. object (객체)
    2. function (함수)
