테두리 관련 스타일 속성 (border-~)

        [ 표현법 ]

        선택자 {

            * 테두리 스타일 (모양)
            border-위치-style : none/dotted/dashed/solid/double/groove/ridge/inset/outset;

            * 테두리 두께
            border-위치-width : 테두리두께(px);

            * 테두리 색상
            border-위치-color : 색상값; (색상명, rgb, rgba, hsl, hsla, #16진수)

            * 테두리 두께, 스타일, 색상을 한번에 지정 가능
            border-위치 : 두께 스타일 색상값; // 색상값 생략 시 기본적으로 글자색과 통일됨

            * 테두리 모서리를 둥그스름 하게
            border-위치1-위치2-radius : 숫자(px); // 위치 생략 시 사방이 둥그스름해짐

            * 테두리 박스에 그림자 효과
            box-shadow : 가로거리 세로거리 흐림정도 번짐정도 색상;
        }

        - 위치 : top/bottom/left/right
        - 위치1 : top/bottom
        - 위치2 : left/right
