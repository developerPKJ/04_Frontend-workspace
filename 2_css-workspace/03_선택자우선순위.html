선택자 우선순위

- 기본적으로 CSS 는 위에서부터 아래로 차례대로 스타일이 적용됨
- 그런데 동일한 요소를 다양한 선택자로 선택하여 스타일이 부여된 경우에는 "우선순위" 에 따라 스타일이 각각 다르게 적용됨
- 단, 겹치지 않는 스타일 항목은 각각 반영된다!! 

        * 우선순위

        태그선태자 -> 클래스선택자 -> 아이디선택자 -> 인라인스타일방식 -> !important
