목록 관련 스타일 속성 (list-~)

list-style-type : 불릿 기호를 변경시켜줄 때 사용 

        [ 표현법 ]

        * 순서 없는 목록 (ul)

        선택자 {
            list-style-type : disc/circle/square/none;
        }

        * 순서 있는 목록 (ol)

        선택자 {
            list-style-type : decimal/decimal-leading-zero;
            list-style-type : lower-alpha/upper-alpha;
            list-style-type : lower-roman/upper-roman;
        }

순서 없는 목록 (ul)

  • HTML
  • CSS
  • JavaScript
  • jQuery

순서 있는 목록 (ol)

  1. HTML
  2. CSS
  3. JavaScript
  4. jQuery


list-style-position : 불릿기호의 위치를 조정할 때 사용

list-style-image : 불릿기호를 이미지로 적용할 때 사용 

        [ 표현법 ]

        선택자 { 
            list-style-position : inside/outside;
            list-style-image : url("이미지경로");
        }
