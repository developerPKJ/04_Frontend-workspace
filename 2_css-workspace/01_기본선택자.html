CSS 의 기본선택자

선택자란?

- 특정 html 요소를 선택하고자 할 때 사용하는 기능
- 해당 요소를 선택해서 원하는 "스타일" 과 기능을 적용시킬 수 있음


1. 모든(전체) 선택자 (*)

- 이 html 문서의 모든 요소들을 다 선택할 때 사용 

        [ 표현법 ]

        * {
            스타일속성명 : 값;
            스타일속성명 : 값;
            ...
        }

2. 태그 선택자 (태그명)

- 해당 이 html 문서의 같은 태그들을 일괄 모두 선택할 때 사용 

        [ 표현법 ]

        태그명 {
            스타일속성명 : 값;
            스타일속성명 : 값;
            ...
        }

3. 아이디 선택자 (#아이디명)

- 해당 이 html 문서 내에 특정 요소 "단 하나만을" 선택하고자 할 때 사용
- html 문서 내의 요소마다 id 라는 속성을 줄 수 있다!! (이 id 속성은 한 문서 내에서 중복되면 안됨, id 속성값으로 요소를 선택하면 자연스럽게 하나만 선택된다) 

        [ 표현법 ]

        #아이디명 {
            스타일속성명 : 값;
            스타일속성명 : 값;
            ...
        }

4. 클래스 선택자 (.클래스명)

- 해당 이 html 문서 내에 내가 원하는 요소 여러개를 한번에 선택하고자 할 때 사용
- html 문서 내의 요소마다 class 라는 속성을 줄 수 있다!! (이 class 속성은 한 문서 내에서 여러번 중복되도 됨, class 속성값으로 요소를 선택하면 자연스럽게 여러개가 선택된다)
