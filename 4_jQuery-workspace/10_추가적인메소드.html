추가적인 메소드

* each 메소드

- 배열의 모든 인덱스에 순차적으로 접근하고자 할 때 또는 객체가 가지고 있는 모든 속성에 순차적으로 접근하고자 할 때 사용하는 for in 문 과 유사하게 수행되는 메소드 (즉, 반복문 역할) 

        [ 표현법 ]

        1) 
        $.each(객체명/배열명, function(매개변수1, 매개변수2) { 
        
            반복적으로 실행할 구문; 
        });

        2) 
        배열명.each(function(매개변수1, 매개변수2) {

            반복적으로 실행할 구문;
        });

        - 2번 표현법은 배열만 가능!!
        - 순차적으로 객체 또는 배열의 속성명 또는 인덱스에 접근할 때 마다 안쪽 익명함수가 실행되는 구조
        - 이 때, 매개변수1 과 매개변수2 는 생략 가능

        만약 배열명을 제시했다면 
        매개변수1 에는 순차적으로 접근된 배열의 인덱스수가 넘어옴
        매개변수2 에는 순차적으로 접근된 배열의 해당 인덱스값이 넘어옴

        만약 객체명을 제시했다면
        매개변수1 에는 순차적으로 접근된 객체의 속성명 (키값) 이 넘어옴
        매개변수2 에는 순차적으로 접근된 객체의 속성에 대한 속성값 (밸류값) 이 넘어옴
