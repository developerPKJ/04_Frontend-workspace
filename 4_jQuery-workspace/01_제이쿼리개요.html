jQuery 개요

* jQuery 란?

- 기존에 자바스크립트 언어 만으로 구현하기에 복잡했던 구문 작성들을 "간소화" 하기 위해 존 레식에 의해 개발된 JavaScript 라이브러리 (유용한 코드 기능들의 모음집)
- 즉, JavaScript 언어로 유용한 함수 또는 내용들을 이미 만들어둔, 정의해둔 코드 모음집

"적은 양의 코드로 빠르고, 풍부한 기능을 제공하겠다."

- jQuery 장점
1) DOM 요소와 관련된 스크립트를 보다 쉽게 구현할 수 있다.
2) Ajax (비동기식 요청 방식), 이벤트 처리 등 폭 넓은 기능을 지원한다.
3) jQuery 와 관련된 확장형 플러그인, 오픈 API 등을 지원한다.
(차트, 슬라이드, 애니메이션 효과, 드래그 앤 드롭 효과 등)


- jQuery 단점
1) 라이브러리가 다소 무거운 편
(최근 4버전이 나오면서 점점 경량화 하려는 움직임이 있음)



* jQuery 라이브러리 연동 방법

- 기본 연동 원리가 JavaScript external (외부) 방식임!!
(외부 .js 파일에 코드를 기술하고 script 태그로 html 에서 가져다 쓰는 방법)
- html 문서의 head 태그 내부에 script 태그로 jQuery 라이브러리 .js 파일을 가져오기!!
- 단, 오프라인 / 온라인 방식이 있음!!

- jQuery 공식 사이트 : 공식사이트

- jQuery 라이브러리는 크게 3가지 버전의 .js 파일을 제공해준다.
1) Compressed 버전 : .min.js 파일 형식으로 배포되는 버전
불필요한 주석, 들여쓰기, 띄어쓰기 등을 최소화 해서 .js 파일의 용량을 줄여서 제공하는 파일 (주로 배포 시 사용함)
2) Uncompressed 버전 : .js 파일 형식으로 배포되는 버전
개발자 같은 사람들이 보기 쉽도록 주석, 코드 정렬 등을 활용하여 가시성을 유지하는 파일 (주로 개발 단계에서 사용함)
3) Slim 버전 : .slim.min.js 파일 형식으로 배포되는 버전
ajax, effect 효과 관련 코드를 아예 뺀 버전의 파일
(우리 수엽때는 사용하지 않을 예정임!!)

- 연동방법 (오프라인 / 온라인)
1) 오프라인 방식 : .js 파일을 내가 직접 다운로드 받아서 내 로컬 컴퓨터에 저장한 후 해당 저장된 파일 경로를 제시해서 연동하는 방법
즉, .js 파일 실물을 내가 직접 보유하는 방법
2) 온라인 방식 : .js 파일을 내가 직접 다운로드 받는게 아니라, 라이브러리 소스코드 파일을 제공하는 웹사이트의 url 주소를 제시해서 가져다 쓰는 방법
즉, .js 파일 실물을 내가 직접 갖고있지 않는 방법
CDN (Content Delievery Network) 방식이라고도 부른다!!



* jQuery 구문 작성하기

- jQuery 구문으로 DOM 요소들을 다루는 구문을 기본적으로 작성할 것이기 때문에, jQuery 명령은 html 문서상의 요소들이 브라우저 창에 다 로딩 된 이후에 실행되어야한다!!

- 순수 자바스크립트에서의 해당 내용은 

            window.onload = function() {
                
                실행할 내용;
            };

            onload 함수 : html 페이지가 브라우저 상에 다 로딩 되면 (다 로딩 된 이후에)
                          자동으로 단 한번만 딱 실행되는 메소드 속성
                          단, 해당 html 문서 내부에서 단 한번만 작성 가능!!
                          해당 문서의 DOM 요소 뿐만 아니라 연동되어있는 외부 파일들까지
                          다 로딩이 된 후 실행됨

- jQuery 에서의 해당 내용은 3가지 구문으로 제공 
            1) 
            jQuery(document).ready(function() {

                실행할 내용;
            });

            2) 
            $(document).ready(function() {

                실행할 내용;
            });

            3) 
            $(function() {

                실행할 내용;
            });

            jQuery 에서의 해당 function 들은
            브라우저 상에 해당 html 문서의 DOM 요소들이 다 로딩되면
            자동으로 곧바로 단 한번 실행됨!!
            한 html 문서 내부에서 "여러번" 기술 가능함!! (제한 없음)

            - jQuery == $
