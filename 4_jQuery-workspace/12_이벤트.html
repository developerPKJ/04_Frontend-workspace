이벤트 핸들러 jQuery 연결 방법 종류

방법 1. 이벤트 메소드를 통한 연결

        $("선택자").이벤트메소드() { ... };

        - 이벤트 메소드 종류
        click() : 클릭 이벤트
        dblclick() : 더블 클릭 이벤트
        mouseenter() : 마우스가 요소에 진입할 때 발생하는 이벤트
        mouseout() : 마우스가 요소에서 떠날 때 발생하는 이벤트
        mouseleave() : 마우스가 요소에서 떠날 때 발생하는 이벤트
        ...

클릭해보세요



방법 2. on() 메서드를 통한 연결

        $("선택자").on("이벤트명", function() { ... });

        - 이벤트명은 문자열로 전달

클릭해보세요

방법 3. 이벤트 위임을 통한 연결

        $("부모 요소").on("이벤트명", "자식 요소 선택자", function() { ... });

        - 부모 요소에 이벤트 핸들러를 연결하고, 자식 요소에서 이벤트가 발생했을 때
            핸들러가 실행되도록 하는 방식
        - 동적으로 생성된 요소에도 이벤트 핸들러가 적용됨

키보드 관련 이벤트

입력 양식 관련 이벤트

focus : 요소에 포커스가 맞춰질 때 발생하는 이벤트 blur : 요소에서 포커스가 벗어날 때 발생하는 이벤트 change : 입력한 데이터의 내용이 변경된 후 포커스가 벗어날 때 발생하는 이벤트
