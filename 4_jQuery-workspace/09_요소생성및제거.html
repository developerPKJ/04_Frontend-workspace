* 요소 삽입 관련 메소드 part1

- 선택된 요소를 "기준" 으로 새로이 생성된 요소를 추가해주는 메소드

$(A).append(B) : A 요소 내의 뒷부분에 B 를 추가 (자손)
$(A).prepend(B) : A 요소 내의 앞부분에 B 를 추가 (자손)
$(A).before(B) : A 요소 앞부분에 B 를 추가 (동위)
$(A).after(B) : A 요소 뒷부분에 B 를 추가 (동위)

* 요소 삽입 관련 메소드 part2

- 선택된 요소가 "삽입" 되는 메소드

$(B).appendTo(A) : B 를 A 요소 내의 뒷부분에 추가 (자손)
$(B).prependTo(A) : B 를 A 요소 내의 앞부분에 추가 (자손)
$(B).insertBefore(A) : B 를 A 요소 앞에 추가 (동위)
$(B).insertAfter(A) : B 를 A 요소 뒤에 추가 (동위)

* 요소객체 복제 메소드

let 변수 = $("선택자").clone(true / false);

- 선택된 요소를 복제해서 반환해주는 메소드
- true / false : 복제할 때의 옵션, 생략 가능 (생략 시 기본값은 false)
이벤트를 복사해올거냐 말거냐에의 차이 (true : 이벤트 복사)

* 요소 객체 제거 및 잘라내기 메소드

- $("선택자").empty();
: 선택된 요소의 모든 하위 요소들을 제거시켜주는 메소드
- let 변수명 = $("선택자").remove()/.detach();
: 그냥 호출구문만 작성하면 선택된 요소 자체를 제거해주는 메소드로 사용 가능
앞에 변수 대입구문과 함께 작성하면 요소를 제거할 때 기본적으로 제거된 요소를 반환해주는 메소드이기 때문에 잘 담아뒀다가 가져다 쓸 수 있게됨!! (잘라내기 효과)
.remove() : 잘라내기 시 기존의 이벤트를 안가져옴
.detach() : 잘라내기 시 기존의 이벤트를 가져옴
